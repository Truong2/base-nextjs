# NextJS

This is a [NextJS-13](https://nextjs.org/) project bootstrapped with `create-t3-app`.

## Structure

- App: NextJS App layout
- Assets: includes assets like fonts, images, svgs
- Components: includes UI components
- Constants: includes constants variables
- Lib: includes code for any external libraries
- Services: includes services like API service, ...
- Styles: includes global CSS
- Types: includes type files
- Utils: includes utils functions files
